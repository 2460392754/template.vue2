# template.vue2

vue2 开发模板

## 安装依赖

```
yarn install
```

### 开发环境运行

```
yarn serve
```

### 项目打包

```
yarn build
```

### 代码校验

```
yarn lint
```

### 代码格式化

```
yanr format
```

### 生成 dll，加快打包编译

```
yanr build:dll
```

### 校验 => 格式化 => 打包

```
yarn build:auto
```
